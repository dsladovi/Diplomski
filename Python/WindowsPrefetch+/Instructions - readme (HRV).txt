prefetch.py skirpta radi na principu da parsa prefetch fileove.
Kod pokretanja skripte u CMD-u koristite opciju -h za help (opcije koje skripta nudi)

-d (direktorij)
-f (single file)

Npr.
	python prefetch.py -d C:\Windows\Prefetch\ 
		na kraju se mora dodati backslash jer skripta nece raditi
	python prefetch.py -f CHROME.EXE-D999B1BB.pf
