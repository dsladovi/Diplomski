prefetch.py is parsing prefetch files from a PC or preallocated folder
when starting a script in command prompt, use -h option for help

-d (directory)
-f (single file)

for example:

	python prefetch.py -d C:\Windows\Prefetch\
		at the end add backslash " \ ". otherwise the script will not work

	python prefetch.py -f CHROME.EXE-D999B1BB.pf
